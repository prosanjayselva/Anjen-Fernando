# QA and Acceptance Criteria

## Functional Acceptance
- All required pages and sections exist
- Both enquiry forms submit successfully
- 3 podcast videos embedded and playable
- Portfolio and testimonials display correctly

## UX Acceptance
- Mobile responsive across common sizes
- Accessible forms with labels and keyboard support
- Clear CTA on every page

## Technical Acceptance
- No TypeScript build errors
- API validation active for form endpoints
- Basic performance budget respected for landing pages
